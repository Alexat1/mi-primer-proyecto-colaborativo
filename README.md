# Mi Primer Proyecto Colaborativo
## Objetivo del proyecto
Aprender a utilizar las herramientas básicas de Git y GitHub en un entorno de desarrollo.
## Integrante
- Alexandra Caguana
## Herramientas colaborativas investigadas
- Git (Control de versiones local)
- GitHub (Plataforma de alojamiento en la nube)
- Figma (Colaboración en diseño de interfaz - UI/UX)Antes de programar en Git/GitHub, los equipos deben diseñar cómo se verá la aplicación. Figma permite a diseñadores y programadores trabajar juntos en tiempo real.
